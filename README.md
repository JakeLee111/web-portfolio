# Web Portfolio

## Project Overview
This web portfolio showcases my skills, projects, and experiences as a web developer. It serves as a personal brand and helps potential employers understand my capabilities and projects in one place.

## Features
- Responsive design for all devices
- Display of recent projects with descriptions and links
- About section to introduce myself and my background
- Contact form to reach out directly
- Live demo of projects

## Technologies Used
- **HTML**: For structuring the content
- **CSS**: For layout and visual styling
- **JavaScript**: For interactive elements

## Live Demo
You can view the live demo of this portfolio at: [Live Demo](https://jakelee111.github.io/web-portfolio/)  

## Local Setup
To set up this project locally, follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/JakeLee111/web-portfolio.git
   ```
2. Navigate into the project directory:
   ```
   cd web-portfolio
   ```
3. Open `index.html` in your preferred web browser.

Feel free to customize and enhance this portfolio according to your needs!
